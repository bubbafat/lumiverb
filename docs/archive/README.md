# Archive

Completed or obsolete ADRs and docs live here.

These documents are kept for historical reference — they describe decisions and designs that have been fully implemented or superseded. They should not be treated as current specifications.

When moving a document here, add a note at the top:

```
> **Archived [date]** — [reason, e.g. "Fully implemented" or "Superseded by ADR-012"]
```
